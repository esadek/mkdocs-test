# ID Stitching dbt Package Docs

## Installation

Clone repository and change directory:

```bash
git clone
cd
```

Create and activate virtual environment:

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

Install required packages:

```bash
pip install -r requirements.txt
```

## Usage

```bash
mkdocs serve
```
